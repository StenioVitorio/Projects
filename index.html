Um pouco sobre o criador destes projetos

Meu nome é Stenio, tenho 20 anos e desde pequeno já tinha interesse em computadores, programação voltada a jogos e softwares e áreas relacionadas, também sempre achei interessante resolver problemas lógicos e de códigos, o que ocorre com frequencia no desenvolvimento de qualquer coisa que envolva programação, como por exemplo: "por que esta função não funciona?", "como posso criar um software baseado nos requisitos x?", "qual tecnica posso utilizar para deixar esse trecho de código melhor?", dentre outros problemas e obstaculos que se encontra durante a criação de algo. E busco um emprego nesta area mais voltada ao back-end, não só por ser uma área que me interesso mas por ser uma área que eu considero ter habilidade.

Eu realizei um curso tecnico em analise e desenvolvimento de sistemas juntamente do ensino medio e agora estou cursando o mesmo curso porém a nivel superior

Projeto De Faculdade - Aplicação de console em C

Neste projeto foi construido uma simples aplicação de console utilizando a linguagem C, com o objetivo de criar uma versão simplificada de um sistema de gestão de um museu, possibilitando a compra de bilhetes para exibições e o registro de usuarios(visitantes) em um arquivo de texto.

Devido a este projeto ser uma aplicação de console isso limitou a customização visual do projeto, porém não utilizamos de ferramentas auxiliares como frameworks ou bibliotecas para não escapar muito dos limites definidos para a criação deste projeto.
